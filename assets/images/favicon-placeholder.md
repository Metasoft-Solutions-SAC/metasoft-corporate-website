# Favicon Placeholder

Este es un archivo placeholder para el favicon. 

## Para reemplazar el favicon:

1. Crea un favicon de 32x32 píxeles (o múltiples tamaños)
2. Formatos recomendados: PNG, ICO, SVG
3. Reemplaza este archivo con tu favicon real
4. Actualiza las referencias en los archivos HTML si cambias el nombre

## Herramientas para crear favicons:

- [Favicon.io](https://favicon.io/) - Generador online gratuito
- [RealFaviconGenerator](https://realfavicongenerator.net/) - Genera todos los tamaños necesarios
- Photoshop, GIMP, Figma, etc.

## Tamaños recomendados:

```html
<!-- En el <head> de tus archivos HTML -->
<link rel="icon" type="image/png" sizes="32x32" href="assets/images/favicon-32x32.png">
<link rel="icon" type="image/png" sizes="16x16" href="assets/images/favicon-16x16.png">
<link rel="apple-touch-icon" sizes="180x180" href="assets/images/apple-touch-icon.png">
```

## Nota:
Por ahora, el HTML referencia `favicon.png` - reemplázalo con tu favicon real.
